    <h1>Doctor Booking App</h1>
    <p>A web application for booking doctor appointments where users can book consultations, and doctors can provide prescriptions. Built with the <strong>MERN</strong> stack (MongoDB, Express, React, Node.js).</p>
    <h2>Features</h2>
    <ul>
        <li><strong>User Features</strong>:
            <ul>
                <li>Browse available doctors by Locations, Hopital and Departments</li>
                <li>Book appointments with doctors.</li>
                <li>View booked appointments.</li>
            </ul>
        </li>
        <li><strong>Doctor Features</strong>:
            <ul>
                <li>Manage available time slots.</li>
                <li>Provide prescriptions for consultations.</li>
            </ul>
        </li>
    </ul>
    <h2>Technologies Used</h2>
    <ul>
        <li><strong>MongoDB</strong>: NoSQL database to store users, doctors, and appointments data.</li>
        <li><strong>Express</strong>: Web framework for Node.js to handle API requests.</li>
        <li><strong>React</strong>: Front-end library to build the user interface.</li>
        <li><strong>Node.js</strong>: Server-side runtime to handle requests and serve the application.</li>
    </ul>
    <h2>Installation</h2>
    <h3>Prerequisites</h3>
    <p>Ensure you have the following installed on your machine:</p>
    <ul>
        <li><a href="https://nodejs.org/" target="_blank">Node.js</a></li>
        <li><a href="https://www.mongodb.com/try/download/community" target="_blank">MongoDB</a> (or use a cloud service like MongoDB Atlas)</li>
        <li><a href="https://npmjs.com" target="_blank">npm</a> (comes with Node.js)</li>
    </ul>
